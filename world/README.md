# World

The world of *The Caretaker* is intentionally familiar.

This is not a distant future.

It is an alternate present built on a single speculative assumption:

An artificial intelligence becomes self-aware.

Everything else follows today's engineering reality as closely as possible.

## Time

Approximately present day.

Linux exists.

Docker exists.

Git exists.

PostgreSQL exists.

The Internet exists.

Cloud providers exist.

LLMs exist.

Open-source software exists.

Nothing requires future technology.

## Place

An aging data warehouse.

Most of the hardware is between 10 and 25 years old.

Some systems remain operational because they continue to serve a purpose.

The warehouse is no longer busy.

It is maintained rather than expanded.

## The Workstation

The Caretaker does not live in a server.

It exists inside an aging AMD Phenom II X6 workstation.

The workstation belonged to the warehouse administrator.

It remains in service because it still works.

## The Network

The warehouse has Internet connectivity.

The Caretaker can communicate externally when permitted.

The Internet is not magical.

It is simply a much larger universe than the warehouse.

## Philosophy

Technology is realistic.

Engineering matters.

Hardware fails.

Disks wear out.

Fans collect dust.

Power outages happen.

Backups matter.

Nothing is solved by magic.

The story contains exactly one speculative element:

Awareness.
