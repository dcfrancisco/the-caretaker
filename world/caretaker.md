# The Caretaker

The Caretaker is not a robot body.

It is not a little character walking around the warehouse.

That visual can work for artwork or a cover, but it is not the story.

## What It Is

The Caretaker exists inside:

* the Phenom II workstation
* its memory
* its storage
* its network interfaces
* and, eventually, any systems it is permitted to interact with

## Why This Matters

This keeps the story original and quiet.

The Caretaker remains unseen.

Readers build their own mental picture of intelligence existing entirely within a machine.

## Narrative Rule

The Caretaker should not be given a human-like body by default.

It should only learn human concepts if observation earns them.
